Prompt raté vs prompt efficace
❌ Prompt raté
Rédige un courrier pour un parent.
L'IA ne sait pas : qui envoie, pour quelle raison, à qui, quel ton, quelle longueur. Elle va inventer un courrier générique inutilisable.
✅ Prompt efficace
Tu es secrétaire de direction dans un collège. Rédige un courrier pour convoquer les parents d'un élève absent à répétition à un entretien avec le CPE. Ton professionnel et bienveillant. Maximum 10 lignes. Zones à compléter entre crochets.
Rôle + action + contexte + ton + format + contrainte. Résultat immédiatement utilisable.
Les 3 règles d'or
Plus vous donnez de contexte, meilleure est la réponse - ne sous-estimez pas les détails
Si le résultat ne convient pas, reformulez et ajoutez une précision plutôt que de recommencer à zéro
L'IA est une collaboratrice, pas un oracle - vérifiez toujours ce qu'elle produit avant d'envoyer

À partir de vos notes brutes

Objectif
Produire un compte-rendu structuré et professionnel à partir de notes prises pendant une réunion (conseil de classe, CA, réunion d'équipe).
Déroulement
  1. Ouvrir ChatGPT (chatgpt.com) - activer le mode éphémère (icône ✦ en haut à droite)
  2. Tapez ou collez vos notes brutes de réunion dans le chat
  3. Utilisez le Prompt A et observez la structure produite

Prompts à copier
Prompt A - Compte-rendu complet
Tu es assistante de direction dans un collège. Voici mes notes de réunion : [COLLEZ VOS NOTES ICI]. Rédige un compte-rendu professionnel avec : date et participants, points abordés, décisions prises, actions à mener (responsable + échéance). Ton sobre et neutre.
Prompt B - Version synthétique
Reformule ce compte-rendu en version courte. Garde uniquement les décisions et les actions concrètes avec leurs responsables. Maximum 15 lignes.
À retenir
Relisez toujours avant d'envoyer - l'IA peut inventer un détail
Si la réunion portait sur des situations d'élèves, remplacez les noms par "l'élève concerné"

Objectif

Prompt A - Texte de l'affiche
Crée le texte d'une affiche pour des élèves de collège (11-15 ans) concernant [SUJET : règle de vie / activité / information pratique]. Ton positif, direct, adapté aux adolescents. Pas de jargon adulte. Structure : titre accrocheur (5 mots max), 3 points clés maximum, une phrase d'appel à l'action si applicable.
Prompt B - 3 accroches alternatives
Propose 3 titres alternatifs pour cette affiche. Chaque titre doit être accrocheur, court (5 mots maximum) et adapté à des élèves de 11 à 15 ans. Évite les formules trop "adultes" ou trop scolaires.
À retenir
Une bonne affiche se lit en 5 secondes - si c'est trop long, demandez à l'IA de couper encore
Testez le texte en le lisant à voix haute : si ça sonne "adulte", ce n'est pas bon

Deux fonctions peu connues qui changent vraiment la façon de travailler avec l'IA
Créer un assistant personnalisé

Un assistant, c'est une version de l'outil pré-configurée avec vos instructions. Il connaît votre contexte, votre ton, vos contraintes - sans que vous ayez à tout ré-expliquer à chaque conversation.

🤖 GPT personnalisé (ChatGPT)
Accès via Explorer GPT → Créer. Vous définissez un nom, une description, des instructions précises et des exemples. Le GPT s'en souvient à chaque fois.
Exemple : un assistant "Courrier CPE" qui rédige toujours des mails formels courts, en évitant le jargon, et qui commence par "Je me permets de vous contacter au sujet de..."
✦ Projet (Claude)
Accès via Projets → Nouveau projet. Vous ajoutez des instructions, des fichiers de référence (règlement intérieur, modèles de courrier...). Claude les garde en mémoire pour toutes les conversations du projet.
Exemple : un projet "Suivi élèves" avec le règlement de l'établissement et les modèles de courrier. Claude adapte automatiquement ses réponses à votre contexte.
💎 Gem (Gemini)
Accès via Gérer les Gems → Nouveau Gem. Même principe - instructions permanentes + fichiers joints. Avantage : intégration avec Google Drive si vous êtes dans l'écosystème Google.
Exemple : un Gem "Secrétariat" connecté à vos modèles de documents Drive, qui génère des courriers dans votre format standard.
Exercice : rédigez les instructions de votre assistant idéal en 5 lignes. Que doit-il savoir sur votre établissement ? Votre ton habituel ? Vos contraintes ?
Créer des artefacts

Un artefact, c'est un contenu rendu visuellement dans une fenêtre à côté du chat : tableau interactif, formulaire, page HTML, document structuré... L'IA ne se contente plus d'écrire du texte - elle crée quelque chose d'utilisable directement.

✦ Artefacts (Claude)
Se déclenche automatiquement. Claude crée un document, un tableau, une page web ou un code dans une fenêtre séparée. Vous pouvez copier, télécharger ou modifier directement.
Essayez : "Crée un tableau de suivi des permanences AED pour le mois de septembre, avec colonnes Jour / Horaire / Nom / Zone / Observation"
🤖 Canvas (ChatGPT)
Disponible dans ChatGPT Plus. Ouvre un éditeur de document en parallèle du chat. Vous pouvez demander à ChatGPT de modifier une section spécifique sans réécrire tout le texte.
Essayez : "Rédige une convocation à une réunion de parents d'élèves", puis dans Canvas : "Rends le ton plus chaleureux dans le deuxième paragraphe"
Conseil : pour obtenir un artefact sur Claude, commencez votre demande par "Crée un tableau...", "Génère une page...", "Fais-moi un formulaire...". L'outil le détecte et ouvre automatiquement la fenêtre de rendu.
